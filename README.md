## Technologies Used

* _HTML_
* _CSS_
* _JavaScript_
* _Jest_
* _Babel_

## Description
This project will caluclate the inputted amount in USD and convert it to the user's choice of currency. 

## Setup/Installation Requirements

* _Clone to your desktop_
* _Open terminal_
* _Run_: `npm install`
* _Run_: `npm run build`
* _Run_: `npm run start`
